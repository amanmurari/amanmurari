🏆 Featured Project

Excel AI Agent

I shipped an AI Agent that runs inside Microsoft Excel.

The problem was simple:
people spend too much time cleaning data, analyzing data, creating charts, making reports, and extracting data manually.

The Excel AI Agent helps users:

- 📊 Analyze data automatically
- 🧹 Clean messy data
- 🧠 Do feature engineering
- 📈 Create charts and visualizations
- 📝 Generate reports
- 📄 Extract data from PDFs
- 🌐 Extract data from web pages
- ⚡ Work directly inside Microsoft Excel

Goal:
Turn Excel into an AI-powered data workspace.
